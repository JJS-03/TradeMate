# TradeMate

Java & Spring Boot로 구현한 중고 거래 플랫폼 백엔드 서버로,  
리눅스 환경에 직접 배포·운영하며 인증, 트랜잭션, 상태 관리를 구현한 프로젝트입니다.

## Project Goal

TradeMate는 중고 거래 과정에서 발생하는  
사용자 권한, 거래 상태 변경, 동시성 문제를 고려하여  
실제 서비스 환경을 가정하고 설계한 백엔드 서버 프로젝트입니다.

## Key Features

- JWT 기반 회원 인증 및 권한 관리
- 상품 등록, 수정, 삭제 및 조회
- 거래 상태(SELLING → RESERVED → SOLD) 관리
- 트랜잭션을 고려한 거래 처리
- 리눅스 환경에서의 서버 배포 및 운영

## Database Design (ERD)

- User, Item, Trade 총 3개의 테이블로 구성
- 사용자와 상품은 1:N 관계
- 상품과 거래는 1:1 관계
- 거래 상태 변경은 트랜잭션으로 처리
